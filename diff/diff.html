Usage:
  • Click Generate to create 2 random arrays of sorted strings.
  • You can also upload your own files instead of generating strings. Or paste into the respective textareas
  • Click save to download a textfile of the contents of the textarea above.
  • String Length: The length of each generated string.
  • Number of Strings: The number of strings generated. Default is 100.
  • Chacter Range: ASCII 97(a) - 97+Range charcters. Example Range of 25 is a-z. Range of 3 is a-c.
  • Source in .js
  • Tests using Jasmine
